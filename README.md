# WebScraper Jobs 🔍

A web scraper built with Node.js, Puppeteer and TypeScript that fetches remote developer job listings from GetOnBoard and displays them in a clean searchable interface.

## Features
- Real-time job scraping from GetOnBoard
- Keyword-based search
- Clean card-based UI

## Tech Stack
- Node.js + TypeScript
- Puppeteer
- Express
